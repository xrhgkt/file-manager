# Файловий менеджер

## Опис проєкту
Цей проєкт — це консольний файловий менеджер, розроблений на мові C++ із використанням C++17. Програма реалізує програмний інтерфейс для взаємодії з файловою системою, забезпечуючи надійне керування даними, відображення метаданих та стійкість до програмних збоїв.


---

## Ключові завдання та функції

Іменування та ідентифікація: Можливість створювати та звертатися до об'єктів за їх логічними іменами.

Логічне моделювання: Трансляція фізичної організації сховища у зручну ієрархічну структуру (файли та директорії).

Керування параметрами: Вилучення та відображення атрибутів файлів (розмір, тип, час останньої модифікації), необхідних для ядра системи.

Стійкість до збоїв: Використання механізмів обробки винятків для запобігання критичним помилкам при некоректних операціях.

Захист даних: Імітація контролю доступу через перевірку існування об'єктів та прав на їх канонізацію.
